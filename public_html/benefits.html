Benefits

Standard Language for Multiple Platforms

Swift was built to be a robust language capable of working across all of Apple's platforms and integrating seamlessly with their frameworks. Since Swift is open-source, is it free to download for everyone and has a strong community pushing the language's progression forward. Furthermore, Apple is backing the language and there are no signs of slowing down for new features and improvements to the language. On top of this, Apple makes their IDE (Xcode) available to download for free.

Learning Swift

Because of its intuitive, clear coding style, Swift is easy to pick up and learn for beginners and advanced programmers alike. Swift was influenced by a lot of popular languages. Because of this, Swift has many of the popular features you'd expect from these languages.

I/O

Swift provides a comprehensive suite of File and Socket I/O through its predefined set of classes and APIs. Some of its build in File I/O Classes are:

  • NSFileManager
  • NSFileHandle
  • NSData
  • NSOutputStream
  • NSInputStream
Socket I/O is provided through the Socket.IO library.

Influencers of Swift

Swift carried over a lot of really cool features from a wide variety of languages. Some of the main influencers are:

  • Objective-C
  • Rust
  • Haskell
  • Ruby
  • Python
  • C#
  • CLU
  • Java
  • Javascript

Features from Other Languages

Some of the features included from other languages are:

  • Generics from Java (templates from C++)
  • New Data Types (ex: Dictionaries and Inferred Data Types
  • Protocols (interfaces from Java)
  • Closures from Javascript
  • Anonymous functions (from C# or Javascript)
  • Optional semicolons (similar to Python)
  • Advanced Memory Management (from Objective-C)
  • Error Handling (similar to Java or C#)
