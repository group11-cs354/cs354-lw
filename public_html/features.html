Features

Swift is a general purpose programming language designed to support multiple programming paradigms.
Programming paradigms supported by Swift include:

Object-Oriented Programming

Includes features to support and extend all aspects of Object-Oriented Programming

  • Inheritance
  • Polymorphism
  • Data Encapsulation
  • Modularization

Functional Programming

  • Immutability using the 'let' keyword
  • First Class Functions
  • Pure Functions (No Side Effects)
  • Filter/Map/Reduce Functionality

Imperative Programming

  • Order of Execution matters

Protocol/Interface-Oriented Programming:

  • Communication between objects through clearly defined messages, returns, and parameters

Concurrent Programming:

  • Threading using Objective-C Foundation framework ( E.g: NSThread )
  • Background Tasks

Safe and Secure

Swift is a safe and secure systems programming language. It provides support for, and enforces, strong typing for objects.
It is hard to encounter NULL references because of the design of the language.

Compact and Clear

The language itself is very compact and has functionalities encapsulated in libraries or frameworks.

Powerful and Modern

Swift has many modern features built in, such as:
  • Closures
  • Pattern-matching
  • Tuples
  • Multiple-returns
  • Autonomous Memory Management using Automatic Reference Counting (ARC)
  • Seamless Interoperability with Objective-C and Cocoa

Performance

Swift is a compiled langauage and is designed to be fast. It uses LLVM as its backend with a modified Clang (Swift-Clang) frontend as its compiler.

Robust I/O

Swift provides a comprehensive suite of File and Socket I/O through its predefined set of classes and APIs. Some of its built in File I/O Classes are:

  • NSFileManager
  • NSFileHandle
  • NSData
  • NSOutputStream
  • NSInputStream
Socket I/O is provided through the Socket.IO library.
