Features

Multi-Paradigm Language

Swift is an object-oriented langauge that also provides features to include support different types of programming. Examples include:

Functional Programming

  • Immutability using the 'let' keyword
  • First Class Functions
  • Pure Functions (No Side Effects)
  • Filter/Map/Reduce Functionality

Imperative Programming

Protocol/Interface-Oriented Programming:

  • Communication between objects through clearly defined messages, returns, and parameters

Concurrent Programming:

  • Background Tasks

Safe and Secure

Swift is a safe and secure systems programming language. It provides support for, and enforces, strong typing for objects.
It is hard to encounter NULL references because of the design of the language.

Compact and Clear

The language itself is very compact and has functionalities encapsulated in libraries or frameworks.

Powerful and Modern

Swift has many modern features built in, such as:
  • Closures
  • Pattern-matching
  • Tuples
  • Multiple-returns
  • Automatic Memory Management
  • Automatic Reference Counting (ARC)
  • Seamless Interoperability with Objective-C and Cocoa
